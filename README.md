# Проект: Распознавание рукописных букв

Этот проект содержит код и данные для обучения нейронной сети на наборе рукописных английских букв (A-Z).

## Особенности проекта

*   **Большие данные в Git**: Обучающий набор `A_Z_Handwritten_Data.csv` (более 600 МБ) корректно загружен в репозиторий с использованием технологии **Git Large File Storage (LFS)**, что обходит стандартные ограничения GitHub на размер файлов.
*   **Чистый репозиторий**: Файлы виртуального окружения Python и служебный кэш (`venv/`, `.venv/`, `__pycache__/`) не загружаются в репозиторий благодаря правильной настройке файла `.gitignore`. Это делает клон проекта легким и готовым к работе.

### Структура проекта:

.
├── 15HW.py                 # Основной скрипт для обучения модели
├── A_Z_Handwritten_Data.csv # Набор данных (управляется через Git LFS)
├── .gitattributes          # Конфигурация для Git LFS
└── .gitignore              # Правила игнорирования ненужных файлов
